# Gorlock AI

Free, open-source AI chat powered by [Groq](https://groq.com). Deploy to Vercel in 5 minutes.

## Models
- **Compound Beta** — default, Groq's best
- **Llama 3.3 70B** — high quality
- **Llama 3.1 8B** — fastest
- **Mixtral 8x7B** — long context
- **Gemma 2 9B** — Google's model
- **DeepSeek R1** — reasoning

## Deploy to Vercel

1. Get a free API key at [console.groq.com](https://console.groq.com)
2. Push this repo to GitHub
3. Import to [vercel.com](https://vercel.com) → Deploy

Users enter their own Groq key in the UI (stored in browser only).

## Run Locally
```bash
npm install
npm run dev
```
